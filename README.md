# pLawo Project

Welcome to the **pLawo** project! This is a Full Stack application consisting of a Node.js/Express backend and a React.js (Vite) frontend styled with Tailwind CSS.

## Project Structure

- `/backend`: Node.js + Express server.
- `/frontend`: React + Vite frontend setup with Tailwind CSS.

## Getting Started

### Prerequisites

Make sure you have [Node.js](https://nodejs.org/) installed on your machine.

### Backend Setup

1. Navigate to the backend directory:
   ```bash
   cd backend
   ```
2. Install dependencies (already initialized):
   ```bash
   npm install
   ```
3. Start the Express server:
   ```bash
   npm start
   # or node server.js
   ```

The backend server should now be running on `http://localhost:5000`.

### Frontend Setup

1. Navigate to the frontend directory:
   ```bash
   cd frontend
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the frontend development server:
   ```bash
   npm run dev
   ```

The frontend application should now be accessible at `http://localhost:5173`. Happy coding!
